# 🔐 Password Strength Checker

A Python command-line tool that evaluates password strength and provides security suggestions.

## 🚀 Features
- Checks password strength (Weak / Medium / Strong)
- Analyzes:
  - Length
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Provides suggestions to improve password security

## 🛠️ Technologies Used
- Python
- string module
